## Description

This project demonstrates basic data analysis using the Pandas library in Python.

## Features

- Read CSV Dataset
- Display Dataset
- Display First & Last 5 Rows
- Display Dataset Shape
- Display Column Names
- Display Data Types
- Calculate Statistics
- Filter Students by Marks
- Filter Students by Department
- Export Filtered Data into CSV Files
